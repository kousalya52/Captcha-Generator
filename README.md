# 🔐 Captcha Generator

A simple and responsive **Captcha Generator** built using **HTML, CSS, and JavaScript**. This project generates a random 6-character captcha and validates the user's input to ensure correct verification.

## 🚀 Features

* Random 6-character captcha generation

* Captcha validation
* Success and error messages
* Modern and responsive UI
* Works on desktop, tablet, and mobile devices

## ⚙️ How It Works

1. A random captcha is generated automatically when the page loads.
2. Click the refresh icon to generate a new captcha.
3. Enter the captcha in the input field.
4. Click **Submit Captcha**.
5. The application checks whether the entered captcha matches the generated one.
6. A success or failure message is displayed.

## 💻 Installation

1. Clone the repository:

```bash
git clone https://github.com/kousalya52/captcha-generator.git
```

2. Navigate to the project folder:

```bash
cd captcha-generator
```

3. Open `index.html` in your browser.
